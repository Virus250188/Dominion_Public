# App Catalog

Pre-built Enhanced Apps for the Dominion Dashboard. Download the ZIP, upload via **Settings > Community Apps**, done.

---

## Available Apps

| App | Category | Sizes | Description | Download |
|-----|----------|-------|-------------|----------|
| [Emby](emby/) | Media | 1x1, 2x1, 2x2 | Live streams, library stats, poster carousel | Built-in |
| [Home Assistant](home-assistant/) | Automation | 1x1, 2x1 | Entity display with picker, custom labels, status dots | [ZIP](home-assistant/home-assistant.zip) |

### Capabilities

| App | Notification Ready | AI Ready | Entity Picker | Widget | OAuth |
|-----|:--:|:--:|:--:|:--:|:--:|
| Emby | - [ ] | - [ ] | - [ ] | - [x] | - [ ] |
| Home Assistant | - [ ] | - [ ] | - [x] | - [x] | - [ ] |

> **Notification Ready** — App can send notifications to the dashboard notification panel.
> **AI Ready** — App provides context for the AI chat assistant.
> **Entity Picker** — App supports selecting individual entities (sensors, devices, etc.).
> **Widget** — App includes a rich widget view for 2x1/2x2 tiles.
> **OAuth** — App uses OAuth for authentication.
